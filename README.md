# Secure_Login
A secure web application built using Node.js, Express.js, and MongoDB that provides user registration and login functionality with password hashing and session management.
## Features
- User Registration & Login
- Password Hashing using bcrypt
- Input Validation
- Session Management
- Logout Functionality
- Protection against SQL Injection
## Technologies Used
- Node.js
- Express.js
- MongoDB
- Mongoose
- bcrypt
- Express Session
## Installation
```bash
git clone https://github.com/your-username/Secure_Login.git
cd secure-login-system
npm install
```
## Configure Environment Variables
Create a `.env` file:
```env
MONGO_URI=your_mongodb_connection_string
SESSION_SECRET=your_secret_key
```
## Run the Project
```bash
npm run dev
```
or
```bash
node server.js
```
Open in browser:
```txt
http://localhost:3000
```
